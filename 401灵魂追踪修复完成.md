   - 检查 Key 长度是否与预期一致
   - 检查是否有 BOM 或空格

   - 8000: `voice_bridge.py`（已修复）
   - 8001: `phi_proxy_layer.py`（已修复）

3. **服务是否完全重启**:
   - 使用 `kill_python_processes.ps1` 强制关闭
   - 等待 5 秒
   - 重新启动

4. **.env 文件位置**:
   - 确认 `.env` 文件在项目根目录
   - 确认与 `voice_bridge.py` 和 `phi_proxy_layer.py` 同级
