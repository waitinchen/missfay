Diplomat API
MissFay Bridge Module v2.5.0

核心接口說明

本接口專為內部與合作系統對接設計，提供穩定、加密且高效的文字轉語音（TTS）與 AI 對話服務。

POST /api/v1/chat

基礎網址： https://missfay.tonetown.ai

認證方式： 需在 Header 攜帶 X-API-KEY

請求參數 (Request)

參數名 類型 必填 描述
message String 是 傳送給心菲的對話文字內容。
user_id String 否 外部系統的用戶唯一識別碼。

整合範例 (Code Example)

JavaScript
Python
cURL
fetch('https://missfay.tonetown.ai/api/v1/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-API-KEY': 'YOUR_BRIDGE_API_KEY'
    },
    body: JSON.stringify({
        message: '心菲，妳今天過得好嗎？',
        user_id: 'MISSAV_USER_001'
    })
})
.then(res => res.json())
.then(data => console.log('語音網址:', data.audio));

⚠️ 安全注意事項： 橋接密鑰應嚴格保存在服務器後端，切勿暴露在前端代碼中。所有生成的音訊文件將在 600 秒（10 分鐘）後自動從存儲伺服器中物理刪除。
